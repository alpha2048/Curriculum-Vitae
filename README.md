## この先について

### 何がやりたいのか
#### ーサービスグロースエンジニアとして成長する 
わたし自身の最終的な目標としては"サービスをグロースさせられるエンジニア"になることです。  
わたしは特に技術に強いこだわりがあったり、起業したいという強い気持ちがあるわけではありません。  
しかし、自分自身に刺さったサービスをもっと伸ばして、使ってくれる人が増えてくれることがとても楽しいな〜と感じています。

この目標に近づいていくために、特定領域に深い知識を持ちつつ、幅広い知識を持ち合わせるエンジニアになることが必要だと感じています。

現在は主にAndroidの開発をメインに担当しているものの、Android周りの知識もまだまだ足りず苦戦しています。  
自分自身はAndroid開発は楽しいと感じているので、ここは重点的に力をつけていきたいと思っています。  
また、最終的な目標のため、インフラ、DevOps、グロースさせるための解析知識やビジネス知識、チームビルディングなども吸収していかなければならないと感じています。

肩書としてはCTOやテックリード、またはVPoEが近いかなと思いますが、肩書にこだわらず会社の状況や自身の強みや嗜好に合わせて幅広く対応できる人になりたいです。

#### ーワークライフバランスをさらに充実させる  
ワークライフバランスの充実について、年々考えるようになってきました。  
自分自身がオタクであり、オタクコンテンツに関わりつつ、音楽活動もやりつつ、また、エンジニアとしてのスキルアップもやりつつ...  
これらをこなすためには、どのような働き方が良いのか、手がかりを探っています。

### 求める職場環境
- 技術力が高まる環境
  - モダンな技術をしっかり開発に取り入れている
  - 勉強会に参加したり積極的なエンジニアが多い
  - むしろ社内勉強会などが積極的に開かれているとよい
- 成果がきちんと給料に反映される体系
  - 前線で成果を上げているエンジニアに還元されること
  - お金は成長するためにも必要なので、しっかり出してほしい
- エンジニアがストレスなく開発できる環境
  - フレックスを導入し、機能している。コアタイムが早くない
  - リモートワークを柔軟に使うことができる
    - 新型コロナウイルスがあるから、というわけではなく、会社としてきちんとリモートワークが機能している
  - 比較的新しく性能の良いPCをメインで使用できる
  - SlackやGithubなどの仕事しやすいツールを使用している、必要があれば柔軟に取り込んでいく
- 要はエンジニアがちゃんと評価され大切にされる環境を求めてます
